# Генератор отчетов по данным студентов

Этот проект генерирует отчеты по данным о подготовке студентов к экзаменам из CSV-файлов. Основной отчет - медианная сумма трат на кофе по каждому студенту.

## Установка

1. Клонируйте репозиторий: `git clone https://github.com/7box7/generate_reports.git`
2. Установите зависимости: `pip install -r requirements.txt`

## Использование

```
python main.py --files data.csv --report median-coffee
```

- `--files`: Пути к CSV-файлам (можно несколько).
- `--report`: Тип отчета (пока только `median-coffee`).

## Структура проекта

- `main.py`: Основной скрипт. Парсит аргументы, читает данные, вызывает отчеты, выводит таблицу.
- `reports.py`: Модуль с функциями отчетов. Содержит `median_coffee_report` и вспомогательные функции.
- `utils.py`: Утилиты. Функция `read_csv_files` для чтения CSV с поддержкой фильтрации столбцов.
- `test_main.py`: Тесты. Покрывают все функции, интеграционные тесты.
- `tools/benchmark.py`: Скрипт для сравнения производительности версий отчетов.
- `tools/generate_csv.py`: Генератор больших тестовых CSV-файлов.
- `requirements.txt`: Зависимости (tabulate, pytest, pytest-cov).
- `math.csv`, `physics.csv`, `programming.csv`: Примеры CSV-файлов с данными.
